# Text-to-Speech Generator - UI Design

## Design Philosophy
- **Clean & Minimal** - No clutter, just what you need
- **Text Input First** - The textarea is the hero
- **Instant Feedback** - Clear loading states and results
- **Mobile Friendly** - Works on any screen size

---

## Color Palette (Suggested)

```

Background: #FAFAFA (Light gray - clean backdrop)
```

## Summary

The UI is intentionally simple:
1. **One main action** - paste text, generate audio
2. **Minimal choices** - just pick a voice
3. **Clear feedback** - loading, success, error states
4. **Instant gratification** - preview and download immediately

This design can be built with just HTML + CSS + vanilla JS for the MVP, then upgraded to React later.
